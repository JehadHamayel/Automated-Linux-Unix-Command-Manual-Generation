[1mNAME[0m               	 touch - change file timestamps

________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mDESCRIPTION[0m	        Update the access and modification times of each FILE to the current time.
________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mVERSION[0m			touch (GNU coreutils) 8.32
________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mEXAMPLE[0m			rm File.txt > /dev/null 2>&1
			ls File.txt > /dev/null 2>&1
			echo $?
			touch File.txt
			File.txt > /dev/null 2>&1
			echo $?
________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mOUTPUT[0m			output1(echo $?):Positive, output2(echo $?):Zero
