________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mNAME[0m               	 rev - reverse lines characterwise

________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mDESCRIPTION[0m	        The rev utility copies the specified files to standard output, reversing the order of characters in every line.
		        If no files are specified, standard input is read.
		        
		        This utility is a line-oriented tool and it uses in-memory allocated buffer for a whole wide-char line. If the
		        
		        input file is huge and without line breaks than allocate the memory for the file may be unsuccessful.
		        
________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mVERSION[0m			rev from util-linux 2.37.2
________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mEXAMPLE[0m			echo -e "Hello World\\nGood Job" > test.txt
			rev test.txt
________________	_________________________________________________________________________________________________________________________________________________________________________________________________
[1mOUTPUT[0m			dlroW olleH
			boJ dooG
